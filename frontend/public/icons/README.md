# Icons Directory

Place your icon PNG files here:

## Required Icons

1. **create-video-icon.png**
   - Size: 64x64px, 128x128px, or 256x256px (recommended: 128x128px)
   - Style: Simple, clean icon representing video creation/recording
   - Examples: Camera icon, video camera icon, plus/circle with play button
   - Format: PNG with transparent background

2. **my-video-icon.png**
   - Size: 64x64px, 128x128px, or 256x256px (recommended: 128x128px)
   - Style: Simple, clean icon representing video library/collection
   - Examples: Video library icon, folder with video icon, collection icon
   - Format: PNG with transparent background

## Icon Design Tips

- Use transparent backgrounds
- Match the purple/blue theme colors if possible
- Keep designs simple and recognizable
- Ensure icons are visible on dark backgrounds
- Icons will automatically have a purple glow effect applied

## Where to Get Icons

- [Flaticon](https://www.flaticon.com)
- [Icons8](https://icons8.com)
- [Font Awesome](https://fontawesome.com) (export as PNG)
- [Material Icons](https://fonts.google.com/icons) (export as PNG)
- Create custom icons using design tools

## Fallback

If icons are not found, the code will automatically show a styled placeholder with gradient background.
